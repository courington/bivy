Bivy CSS Framework

A basic flexible framework for rapid prototyping or production. Small and simple coverage for use cases from mobile to large screen.

Bivy is an open source CSS framework built on SASS (but is easily portable to LESS)

The intuitive syntax inspired by other larger frameworks helps the developer build out a prototype or production app/site quickly with a light footprint. Using % and Ems, Bivy stays flexible to be easily purpose built for smaller screens or designed for responsiveness across multiple devices.

Bivy is a baseline framework that you can use and/or fork to get started building on top of.
